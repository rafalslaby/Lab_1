# Remote Tic Tac Toe Game

Remote Tic Tac Toe Game with PC

### Prerequisites

Python 3.6

### Installing

None

### Usage

first console:
```
python TicTacToeGame.py
```
second conslose:
```
python Remote_PC.py
```

### Description

My implementation of Tic Tac Toe on two consoles via TCP.
I failed with protobuf so i simplified the connection.
